# oauth

## Project setup
```
yarn install
```

Create `.env` file in root with content:

```txt
VUE_APP_DREDIRECTURI=<redirect uri>
VUE_APP_API=<API URL>
```

note: no trailing /'s.

### Compiles and hot-reloads for development
```
yarn serve
```
